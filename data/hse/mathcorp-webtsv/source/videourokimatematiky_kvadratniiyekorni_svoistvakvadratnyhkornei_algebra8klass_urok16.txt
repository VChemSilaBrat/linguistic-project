text: "Квадратные корни и их свойства. Квадратным корнем из числа а называют число, квадрат которого равен а. Например, если икс в квадрате равняется двадцать пять, то икс, равное пять и икс, равное минус пять – квадратные корни из числа двадцать пять. Арифметическим квадратным корнем из числа а называют неотрицательное число, квадрат которого равен а, то есть если икс в квадрате равняется девять, то икс, равный трем есть арифметический квадратный корень из числа девять. Арифметический квадратный корень из числа а обозначается так, при этом а называется подкоренным выражением. Действие, с помощью которого вычисляют арифметический квадратный корень называется извлечением квадратного корня. Например, корень квадратный из числа двадцать пять равен пяти, при этом, пять является арифметическим квадратным корнем. Корень квадратный из числа восемьдесят один равняется девяти, в этом случае девять также является арифметическим квадратным корнем. Равенство: корень квадратный из а равняется бэ является верным, если бэ – число неотрицательное и бэ в квадрате дает а. При а меньше нуля, то есть при отрицательном значении а, корень квадратный из а не имеет смысла, поскольку квадрат любого числа является числом неотрицательным. Например, корень квадратный из числа минус двадцать пять не имеет смысла. При любом а, если корень квадратный из а имеет смысл, выполняется равенство: корень квадратный из а в квадрате равен а. Например, корень квадратный из девяти в квадрате равняется девяти. Корень квадратный из числа семь в квадрате равняется семи. 
Рассмотрим свойства арифметического квадратного корня. Первое свойство: если а – неотрицательное число и бэ неотрицательное число, то корень квадратный из произведения а на бэ равняется корень квадратный из а, умноженный на корень квадратный из бэ. Например, корень квадратный из четыре, умноженное на девять, равен корню квадратному из четырех, умноженному на корень из девяти. Значение корня квадратного из четырех равняется два, значение корня квадратного из девяти – три, перемножаем числа, получаем шесть. Или корень квадратный из шестнадцать икс – это корень из шестнадцати, умноженный на корень из икс. Корень из шестнадцати можно вычислить, получим четыре. Корень из икс дописывается отдельно. Второе свойство: если а – число неотрицательное, бэ – положительное, то корень квадратный из дроби, в числителе, в знаменателе бэ, равняется дроби, в числителе которой стоит корень из а, в знаменателе корень из бэ. Например, корень из тридцати шести шестнадцатых можно вычислить следующим образом: отдельно извлечь корень из числителя, отдельно из знаменателя. В числителе получим шесть, в знаменателе – четыре. Выполним сокращение дроби на два, получим три вторых или одну целую пять десятых.  Третье свойство: для любого значения а верно равенство: корень из а в квадрате равняется модулю а. Например, корень квадратный из минус трех в квадрате равняется модуль числа минус три. Модуль числа минус три, в свою очередь, равен противоположному ему, положительному числу три или корень квадратный из четыре игрек в квадрате равняется. Будем пользоваться первым свойством, отдельно вычислим корень из четырех, отдельно корень квадратный из игрек в квадрате. Корень квадратный из четырех равен два, корень квадратный из игрек квадрат – модуль игрек. Рассмотрим следующие действия. Первое: вынесение множителя из–под знака корня. Выполняется следующим образом: рассмотрим пример корень квадратный из числа сто двадцать пять. Представим число сто двадцать пять в виде произведения двух чисел, двадцать пять на пять. Вычисляем отдельно корень квадратный из числа двадцать пять и корень квадратный из числа пять. Получаем пять корней из пяти. И второе действие: внесение множитель… множителя под знак корня. Также рассмотрим на примере: десять корней из двух. Нужно число десять внести под знак квадратного корня. Вносим его следующим образом: записываем под знаком корня квадрат числа десять и выполняем действия под корнем. Десять в квадрате равняется сто, сто умножаем на два, получаем двести. Наш ответ: корень квадратный из числа двести. Далее рассмотрим алгоритм решения квадратного уравнения икс в квадрате равняется а. При этом возможны следующие случаи: первый, когда а – число отрицательное. В этом случае уравнение решений не имеет. Действительно, рассмотрим пример: икс в квадрате равняется минус двадцать пять. Поскольку не существует такого числа, которое в квадрате даст минус двадцать пять, следовательно наш вывод о количестве решений является верным. Второй случай, когда а равно нулю. В этом случае, уравнение икс в квадрате равняется нулю имеет одно решение – икс, равное нулю. Единственное решение. И третий случай, когда а больше нуля, в этом случае наше квадратное уравнение имеет два решения: икс первое вычисляется как корень квадратный из числа а, икс второе равняется минус корень квадратный из числа а. Например, икс в квадрате равно сто сорок четыре. Икс первое будет равно корню квадратному из числа сто сорок четыре, то есть двенадцати. Икс второе – минус корню квадратному из числа сто сорок четыре, то есть минус двенадцати. Рассмотрим еще одно уравнение: икс в квадрате равняется семь, в этом случае икс первое равняется корню квадратному из семи, икс второе – минус корню квадратному из семи."
